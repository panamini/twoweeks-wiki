# Documentation et audit de run.sh — 15.2 Recommandations concrètes — Ajouter un fichier de log principal.

Source: `audit/Documentation et audit de run.sh.md`

Paragraphe: 15.2 Recommandations concrètes

Amélioration proposée: Ajouter un fichier de log principal.

Par exemple:
	```text
	tmp/run-dev-stack.log
	```
	Il contiendrait les actions du script lui-même, distinctes des logs applicatifs Vite et Convex.
