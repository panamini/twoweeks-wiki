# Git Good Habit Skill

## Summary

Tiny runbook for starting a new task from an updated `main` branch and then creating a short descriptive branch.

## Key points

- Switch to `main`.
- Pull fast-forward from `origin/main`.
- Create a short descriptive task branch.
- This is a branch-start habit, not a command to run over local uncommitted work.
