# Claude Light Mode Palette

## Summary

Palette extraction note for light and dark UI tokens inspired by Claude-style warm neutral surfaces and terracotta accents.

## Key points

- Light canvas around `#F8F7F3`, header `#FAF9F5`, card `#F0EEE6`, title `#0F0C08`.
- Terracotta/deep clay values include `#A84E2E` and action accent around `#D97757`.
- Dark mode includes near-black canvas and warm neutral text/surface tones.
- Palette should inform UI chrome tokens, not document-layer typography or export geometry.

Supports [[design/elite-design-system]] as palette evidence.
